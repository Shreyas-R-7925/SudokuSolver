# SudokuSolver 


#### * The SudokuSolver application automatically solves Sudoku puzzles when the users uploads the image of Sudoku board to the website. 
#### * Powered by Convolutional Neural Networks (CNNs), the application accurately identifies the digits on the board. 
#### * It uses backtracking algorithm for puzzle solving and delivers the solved Sudoku puzzle image within seconds.
